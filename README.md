# Project Overview

The **AI-Enabled Student Admission System** is a cloud-based Salesforce CRM application designed to automate and streamline the complete student admission lifecycle for educational institutions.

The project combines Salesforce CRM capabilities with automation and Artificial Intelligence to improve admission management, reduce manual effort, maintain data accuracy, and support faster decision-making.

The application provides a centralized platform for managing student information, course details, enrollment processes, approval workflows, communication, reporting, and AI-powered insights using Salesforce and Agentforce AI.

---

# Problem Statement

Traditional admission processes often depend on spreadsheets, emails, and manual documentation, which lead to:

- Duplicate student records
- Data inconsistency
- Delayed approval processes
- Manual communication efforts
- Limited visibility into admission status
- Lack of intelligent decision support

The proposed Salesforce solution addresses these challenges through automation, security, and AI-powered analysis.

# Project Objectives

The major objectives of this project are:

- Develop a centralized student admission management system

- Maintain accurate student and enrollment data
- Implement approval workflows for admission decisions
- Provide secure role-based user access
- Improve communication using automated notifications
- Generate reports and dashboards for better insights
- Integrate Agentforce AI for intelligent recommendations
